# Screenshots Directory

This directory contains screenshots for the main README.md file.

## Required Screenshots

The following screenshots should be captured and placed in this directory:

1. **focus-mode.png** - Focus Mode showing a single task with action buttons
2. **task-list.png** - Task List view with multiple tasks, filters, and columns
3. **priority-comparison.png** - Priority comparison dialog showing two tasks side-by-side
4. **task-form.png** - Task creation/editing form with all fields visible
5. **postpone-dialog.png** - Postpone dialog with reason selection options
6. **dependency-graph.png** - Dependency graph visualization showing task relationships
7. **settings-dialog.png** - Settings dialog showing one of the configuration tabs
8. **notification-panel.png** - Notification panel with recent notifications

## Screenshot Guidelines

- **Resolution**: 1920x1080 or similar high-resolution
- **Format**: PNG (for transparency and quality)
- **Theme**: Capture both light and dark theme versions if needed
- **Content**: Use realistic but clean sample data (avoid personal information)
- **Focus**: Highlight the key feature in each screenshot
- **Annotations**: Add subtle annotations if needed to highlight important UI elements

## Naming Convention

Use lowercase with hyphens for consistency:
- Good: `focus-mode.png`, `task-list.png`
- Avoid: `FocusMode.PNG`, `task_list.jpg`

## Tools

Recommended screenshot tools:
- **Windows Snipping Tool** (Win+Shift+S)
- **ShareX** (free, open-source)
- **Greenshot** (free, open-source)

For annotations:
- **Paint.NET** (free)
- **GIMP** (free, open-source)
